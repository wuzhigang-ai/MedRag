## 场景三: Agent多步推理问答 (1:45-3:15)

### 问题1 (1:45-2:15)
**画面**: 打开 http://localhost:8000/chat 聊天页面
- 输入: "Stanford B型主动脉夹层的诊断标准是什么？"
- 点击发送

**画面重点**: Agent Workbench实时展示推理过程
- 步骤1: search_rag → "检索: Stanford B型主动脉夹层诊断标准" → 显示找到的结果预览
- 步骤2: list_docs → "列出知识库文献"
- 步骤3: cross_check → "交叉验证多文献结论一致性"
- 步骤4: 最终答案生成

**解说**: "Agent自动规划6步推理流程，实时展示工具调用过程。每个步骤的输入参数和输出预览都可视化呈现。"

### 问题2 (2:15-2:45)
**画面**: 
- 输入: "急性TBAD患者应该使用什么降压药物？目标血压是多少？"

**画面**: 
- 格式化的Markdown回答: 含标题层级、表格、药物列表、剂量范围
